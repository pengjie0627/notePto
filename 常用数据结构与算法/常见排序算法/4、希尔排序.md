1、思想
```
1）通过数组的长度/2得到第一趟排序的增量，按照增量进行分组。（通常除以2，也可是其他数值）
2）每个分组使用直接插入排序进行位置的交换
3）通过第一趟排序的增量/2得到第二次排序的增量，向上取整，在按照增量进行分组。
4）每个分组使用直接插入排序进行位置的交换
5）直到得到的增量为1时，进行的直接插入排序的结果为希尔排序的结果。
```
2、js实现
```
function shellSort(arr) {
    var len = arr.length,
        temp,
        gap = 1;
    while(gap < len/3) {          // 动态定义间隔序列
        gap =gap*3+1;
    }
    for (gap; gap > 0; gap = Math.floor(gap/3)) {
        for (var i = gap; i < len; i++) {
            temp = arr[i];
            for (var j = i-gap; j >= 0 && arr[j] > temp; j-=gap) {
                arr[j+gap] = arr[j];
            }
            arr[j+gap] = temp;
        }
    }
    return arr;
}
```
3、时间复杂度
```
最坏的情况：O(n^2)
最好的情况:O(n^1.3)
平均的情况：时间复杂度大约在O(n^1.3)到O(n^2)之间
```
4、稳定性
```
非稳定，间隔可以设置不同
```
5、应用场景
```
基本有序的数据：当数据已经基本有序或接近有序时，希尔排序的效率较高。
```
