和vue2的区别是：
在属性链接这一块不再使用Object.defineProperty，而是使用Proxy+Reflect，用法如下：
new Proxy(catogory,{
    get(target, prop){
        return Reflect.get(target, prop)
    },
    set(target, prop, val){
        return Reflect.set(target, prop, val)
    },
    deleteProperty(target, prop){
        return Reflect.deleteProperty(target, prop)
    }
})
其他的依赖收集和vue2都一样
